# 주식/경제 정보 웹 — 기획 종합 정리

> 이 문서는 새로운 대화를 시작할 때 그대로 붙여넣어 컨텍스트를 이어가기 위한 참고 자료입니다.

## 0. 프로젝트 개요

- **목적**: 국내+해외 주식/경제 뉴스 모니터링, 세계 소식 모니터링, 뉴스 기반 주가 변동성 예측
- **시장 범위**: 국내(코스피/코스닥) + 해외(미국 등) 둘 다
- **API 정책**: 무료 API만 사용 (비용 0원 지향)
- **핵심 철학**: 정보가 항상 떠 있는 게 아니라, 평소엔 가볍고 이상 신호/필요한 시점에만 깊이 들어갈 수 있는 구조 ("조건부 노출")

---

## 1. 화면 구조 (탭 기반)

```
[홈]        시장지표 + 내 포트폴리오(+변동원인 배지) + 오늘의 주목종목 + 다음 중요이벤트 + 뉴스피드
[종목분석]   종목별 영향관계 구조(과거)
[미래이슈]   이벤트 타임라인 + 영향권 + 선행지표(미래)
[스크리너]   재무/배당/PER/주주환원율 필터
```

## 2. 기능별 상세 설계

### 2-1. 홈 화면

- **시장 지표 바**: 코스피/코스닥, 나스닥/S&P500/다우, 환율, 유가·금, **VIX**(시장 심리지표), (선택) 공포-탐욕지수
- **내 포트폴리오/관심종목**: 종목명, 티커, 현재가, 평단가(보유시), 수익률, 변동성 배지
- **오늘의 주목 종목** (사용자 관심종목 외 발굴):
  - 거래량 급증 종목
  - 등락률 상위 종목
  - 뉴스 언급 급증 종목
  - 연관 추천 (내 관심종목과 같은 섹터/공급망인데 안 보고 있는 종목 — *섹터 마스터 데이터 필요*)
- **다음 중요 이벤트**: 내 포트폴리오 연관도/임박도 기준 1개만 노출
- **뉴스 피드**: 국내+해외, 종목/섹터 태그, (확장 시) 긍정/부정/중립 라벨

### 2-2. "왜 움직였는지" 영향관계 (조건부 노출 설계)

**1단계 — 평소엔 숨김, 임계값(예: ±2%) 초과 시만 배지 노출**
```
평소:        삼성전자   71,200   +0.3%
변동 클 때:   삼성전자   68,900   -2.1%  [🔻 반도체 업황]
```
우선순위: 종목고유 > 동종업계 > 섹터 > 매크로 (가장 구체적인 원인 먼저)

**2단계 — 클릭 시 아코디언으로 5단계 구조 펼치기**
```
매크로 → 섹터 → 동종업계(경쟁사) → 수급(외국인/기관, 국내 한정) → 종목 고유 뉴스/공시
```
최대 3~4개 요인까지만 노출, 다시 클릭하면 접힘

**3단계 — 전역 토글**: "변동 원인 분석 [켜짐/꺼짐]" — 끄면 순수 시세창

### 2-3. 미래 이슈 (선행지표 포함)

**1단계 — 홈엔 가장 중요한 다음 이벤트 1개만**
```
📅 D-3   FOMC 금리 발표 → 내 보유종목 중 3개 영향권
```

**2단계 — 클릭 시 3단 구조**
1. 영향권: 어떤 종목/섹터가 영향받나
2. 과거 패턴: 비슷한 이벤트의 과거 변동 사례
3. **지켜봐야 할 선행지표**: 이벤트 전 체크할 신호 (예: CPI 추이, 고용지표, 연준 발언)

**3단계 — [미래이슈] 탭의 전체 타임라인**: 가로 타임라인, 카테고리별 색상(정책/실적/매크로지표/산업), 포트폴리오 연관 이벤트는 ★ 표시

### 2-4. 재무 스크리너

| 항목 | 내용 |
|---|---|
| 재무 상태 | 매출/영업이익/순이익 추이(3~5년), 부채비율, ROE |
| 배당 | 배당수익률, 배당 지속 연수(연속 배당/증액) |
| 밸류에이션 | PER, PBR |
| 주주환원 | (배당+자사주매입)/순이익 |

- 기본 필터: **PER 10 이하 & 주주환원율 40% 이상**, 슬라이더로 사용자 조정 가능
- 국내 종목부터 정확히 구현 → 해외는 점진 확장 (해외 재무데이터는 무료로 깊이 있게 얻기 어려움)

### 2-5. 추가된 데이터 (3차 보완)

| 항목 | 처리 방식 | 갱신 주기 |
|---|---|---|
| 섹터/산업 분류 | 마스터 테이블 (배치) — 동종업계 비교, 연관추천의 전제조건 | 월 1회 |
| VIX | 매크로 지표, 홈 시장지표 바 | 일 1회 |
| 공포-탐욕지수 | 보조 지표(선택), 비공식 소스라 우선순위 낮음 | 일 1회 |
| 수급(외국인/기관 순매수) | 종목별 주기적 수집 + 영향관계 5단계 레이어, **국내 종목 한정** | 일 1회(장마감 후) |

## 3. MVP 범위 결정

| 항목 | 결정 | 이유 |
|---|---|---|
| 개인 매매기록/투자일지 | **제외** (1단계 MVP) | CRUD UI 공수 큼, 본 목적(모니터링/인사이트)과 거리 |
| 알림 기능 | Slack/텔레그램 웹훅으로 **하루 2번 요약 알림** | GitHub Actions는 상시 소켓 연결 불가, 가벼운 배치 알림이 현실적 |
| 수급 데이터 | **포함** (사용자 요청으로 추가) | 국내 가격 영향력이 커서 영향관계 설명에 필요 |
| 섹터 분류 | **포함, 필수** | 없으면 연관추천/동종업계 배지 기능 자체가 작동 불가 |

## 4. "웹 vs 멀티에이전트" 아키텍처 결론

- 웹(UI)과 멀티에이전트(백엔드 로직)는 대체 관계가 아니라 역할이 다른 레이어
- 단순 조회성 정보(시세, 재무수치): API 호출 + 캐시만으로 충분, 에이전트 불필요
- 해석이 필요한 정보(인과관계, 미래 이슈): LLM 호출이 의미 있음 — **단, 임계값을 넘은 "내 관심종목"에만 한정 트리거** (비용 방어)
- "실시간"은 무료 API 기준 주기적 polling이 현실적 (시세 1~5분, 뉴스 10~15분, 재무/분석은 더 느리게)

## 5. 추천 기술 스택

```
[Next.js 프론트엔드]  ← Vercel 무료 배포
        ↑ (DB 조회만)
[Supabase (PostgreSQL)] ← 무료 티어, 캐시/저장소
        ↑ (주기적 적재)
[GitHub Actions 스케줄러] ← public 저장소면 무료 분량 무제한
        ↓
[Python 수집 스크립트]
   - 시세: KIS API, Finnhub
   - 뉴스: 네이버 뉴스 API, RSS
   - 재무: DART Open API
   - 수급: pykrx
   - 분석(인과관계/미래이슈): 필요 시점에만 Claude API 호출
```

**중요 팁**:
- GitHub Actions 무료 분량 제한(월 2,000분)은 **private 저장소에만 적용**. **public 저장소는 무제한 무료** → API 키는 GitHub Secrets에 보관하면 public이어도 안전
- LLM 호출은 "내 관심종목 + 변동성 임계값 초과" 조건을 동시에 걸어 비용 방어
- KIS(실시간) vs Finnhub(지연) 시세 혼재 → UI에 "최소 15분 지연 데이터" 명시 필요

## 6. 데이터 소스 매트릭스 (무료 API 기준, 확인된 정보)

| 데이터 | 후보 | 무료 조건 | 주의점 |
|---|---|---|---|
| 국내 시세 | 한국투자증권 KIS Developers | 무료(계좌 필요) | 가입 시 세부 한도 재확인 |
| 해외 시세(미국) | Finnhub | 분당 60회 무료 | 미국 외 거래소는 유료 필요 |
| 해외 시세(미국 외) | yfinance(비공식) | 무료, 키 불필요 | 비공식, 불안정, 약관 위반 소지 |
| 국내 재무/공시 | DART Open API | 무료(호출제한 있으나 개인 규모엔 충분) | - |
| 해외 재무 | Finnhub 기본 펀더멘털 / yfinance | 무료지만 깊이 제한적 | 상세 항목은 yfinance로 보완 필요 |
| 국내 뉴스 | 네이버 뉴스 검색 API | 하루 25,000회 | 매우 넉넉 |
| 해외 뉴스 | NewsAPI.org | 하루 100회 | 개발환경 전용, 외부 배포 시 차단/약관위반 소지 |
| 해외 뉴스(대안) | RSS 피드(로이터/블룸버그 등) | 무료, 무제한 | 키워드 검색 불가하지만 가장 안정적 |
| 국내 수급 | pykrx | 무료, 키 불필요 | 비공식 스크래핑 기반 |
| 섹터분류 | KRX 업종분류 / DART 기업개황 / Finnhub `finnhubIndustry` / yfinance `info['sector']` | 무료 | 월 1회 배치 갱신 |
| VIX | FRED API | 무료, 한도 매우 넉넉 | - |

참고: Alpha Vantage는 무료 한도가 하루 25회로 줄어 메인 시세 소스로는 부족.
